Gmail API Setup for Southmead Teaching

Step 1: Enter your Google OAuth credentials

From Google Cloud Console → APIs & Services → Credentials → your OAuth Client ID

This will open a Google sign-in window. Sign in with teachsurgerysmh@gmail.com

Step 2: Waiting for authorization...

Complete the sign-in in the popup window, then click "Allow".

Your tokens are ready!

Now add these 3 secrets in Supabase:
Dashboard → Project Settings → Edge Functions → Secrets
• GMAIL_CLIENT_ID
• GMAIL_CLIENT_SECRET
• GMAIL_REFRESH_TOKEN
